# PromovaWeb

A clean and confident theme extracted from promovaweb.com. Combines a vibrant teal brand accent with dark navy CTAs and a crisp white background — ideal for digital marketing, LowCode education, and tech community content.

## Color Palette

- **Brand Teal**: `#00C9A7` - Primary accent, logo highlight, display headlines
- **Teal Dark**: `#00957A` - Hover states, borders, secondary teal elements
- **Navy CTA**: `#1A3A5C` - Primary button background, strong CTAs
- **Heading**: `#111827` - H1/H2 titles
- **Nav / Subtext**: `#374151` - Navigation links, secondary text
- **Body Text**: `#6B7280` - Paragraphs, descriptions
- **Background Light**: `#F9FAFB` - Page/section background
- **Icon Background**: `#EEF2FF` - Icon container fills
- **Borders**: `#E5E7EB` - Dividers, card borders
- **White**: `#FFFFFF` - Card surfaces, hero background

## Typography

- **Display (H1)**: Inter ExtraBold 800
- **Headers (H2/H3)**: Inter Bold 700
- **Body Text**: Inter Regular 400
- **UI Labels / Nav**: Inter Medium 500

## CSS Variables

```css
--pw-teal:       #00C9A7;   /* brand accent, headlines */
--pw-teal-dark:  #00957A;   /* hover, borders */
--pw-navy:       #1A3A5C;   /* primary CTA button */
--pw-heading:    #111827;   /* h1/h2 */
--pw-body:       #6B7280;   /* paragraphs */
--pw-bg:         #F9FAFB;   /* page background */
--pw-border:     #E5E7EB;   /* dividers */
```

## Best Used For

Digital marketing presentations, LowCode and automation product decks, EdTech course landing pages, community onboarding materials, SaaS product showcases for Brazilian market.
